#  Simple Calculator in Python

A basic command-line calculator built using Python functions.  
This project performs four arithmetic operations:

  Addition
  Subtraction
  Multiplication
  Division (with zero division handling)


##  Features

  User-friendly menu
  Function-based structure
  Input validation
  Division by zero handling
  Clean and readable code


##  Technologies Used

Python 3
